# 🚗 Vantage Cars Sales Data Analysis

Project Overview

A data analytics case study examining 1,491 car sales transactions across 6 branches in South-West Nigeria. This project uncovers sales trends, brand performance, customer demographics, and regional patterns to support data-driven business decisions at Vantage Cars.



Objectives

- Analyze sales performance across all 6 regional branches
- Identify top-performing car brands and models
- Understand customer demographics and purchasing behaviour
- Detect regional and seasonal sales trends
- Provide actionable recommendations to improve revenue and branch performance



Tools & Techniques

| Tool | Usage |
|---|---|
| Microsoft Excel | Data cleaning, analysis, pivot tables, dashboards |
| Excel Charts | Data visualization (bar, line, pie charts) |
| Pivot Tables | Multi-dimensional sales breakdowns |
| Conditional Formatting | Trend highlighting and KPI flagging |


Key Features

- Branch Performance Comparison — Side-by-side revenue and unit sales across all 6 locations
- Brand & Model Analysis — Best-selling brands by volume and revenue
- Customer Demographics — Breakdown by age group, gender, and buyer type
- Regional Trend Analysis — Geographic sales patterns across South-West Nigeria
- Interactive Dashboard — Excel-based visual summary with slicers and dynamic charts



Key Insights

- Identified the top-performing branch by both revenue and units sold
- Revealed the most popular car brands among South-West Nigerian buyers
- Uncovered demographic segments driving the highest purchase volumes
- Detected seasonal fluctuations that can inform inventory and staffing decisions
- Highlighted underperforming branches with recommendations for improvement

Dataset Overview

| Field | Description |
|---|---|
| Transaction ID | Unique identifier for each sale |
| Branch | One of 6 South-West Nigeria locations |
| Car Brand / Model | Make and model sold |
| Sale Price | Revenue per transaction |
| Customer Demographics | Age, gender, buyer type |
| Sale Date | Date of transaction |

Recommendations

Based on the analysis, the following actions are recommended for Vantage Cars:

1. Replicate top-branch strategies across lower-performing locations
2. Stock high-demand brands based on regional preferences
3. Target key demographic segments in marketing campaigns
4. Plan inventory around peak sales periods identified in the trend analysis
